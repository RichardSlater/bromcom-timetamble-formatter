## [v0.1.0] - 2025-11-22

### Added
- Initial CLI + core library capable of parsing Bromcom PDFs and rendering SVG timetables with map integration
- Room-to-department mapping with separate background and foreground colors
- Per-week/day/period override system for correcting parsing errors
- A4-sized SVG output with embedded highlighted school maps
- Release and CI workflows with cross-platform builds
- Example configuration and comprehensive documentation
- Community guidelines: CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md, SUPPORT.md
- Pre-commit hooks for code quality enforcement
